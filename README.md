# Commercial Portal Platform

Добро пожаловать в проект **Commercial Portal** — современную платформу, предназначенную для публикации событий, акций и скидок от локального бизнеса, а также для взаимодействия с пользователями через подписки и уведомления.

---

## Возможности

- Регистрация пользователей: бизнес-профили и клиенты
- Публикация событий, акций и скидок бизнесами
- Генерация красивых описаний постов через YandexGPT
- Подписка клиентов на бизнес-профили
- Получение уведомлений о новых публикациях через email
- Управление филиалами и локациями бизнесов с помощью Яндекс Геокодера
- Поиск и фильтрация постов по типу публикации и городу
- Защищённая аутентификация с использованием ASP.NET Core Identity
- Масштабируемая архитектура Vertical Slice + CQRS + Clean Architecture

---

## Технологии

| Блок            | Технологии                                                                 |
|-----------------|-----------------------------------------------------------------------------|
| Backend         | .NET 8, ASP.NET Core, Entity Framework Core, MediatR, FluentValidation, CSharpFunctionalExtensions |
| Frontend        | React + TypeScript, React Router                                            |
| Интеграции      | YandexGPT API, SMTP (email уведомления)                 |
| Инфраструктура  | Docker (контейнеризация), Swagger (документация API)                         |
| База данных     | PostgreSQL 15                                                                |


## Структура проекта

/CommertialPortal_WebAPI /API # Контроллеры Web API /Application # Интерфейсы и контракты /Domain # Бизнес-сущности и перечисления /Features # Вертикальные срезы (CQRS Handlers) /Infrastructure # Реализация сервисов (Email, Token, Geocoder) /Migrations # Миграции базы данных
